# OOM Smart Pot
This is a smart pot system for the class Practical development for Embedded and IoT systems in University of Tsukuba as the final design.  
このリポジトリは筑波大学の「組込みプログラム開発」という授業における、最終デザインとして作成されたスマトポットシステムです。　

# Commit Message
```text
<type>[optional scope]: <description>
[optional body]
[optional footer(s)]
```

## Optional Type
- feat
- fix
- build
- docs
- refactor
- test
- style
- perf
- chore

## Body
Detail description of commit.

## Optional Footers
- Test
- Review
- Issue
- Close
- Others

## Example
```text
feat: implement Gemini api call.
Add a function into backend that allow the system to call Gemini API and get insights and suggestions for the plant care.  
Test: Local Passed
```
