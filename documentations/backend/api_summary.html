Smart Planter API Summary

Group Method Endpoint Description
Sensors GET /api/v1/sensors/current Retrieve current or historical sensor telemetry data.
Actuator Control POST /api/v1/pump/action Execute a single manual watering task for a specified duration.
Actuator Control GET /api/v1/pump/status Fetch current real-time electrical state and operational health metrics.
Actuator Control POST /api/v1/pump/stop Immediately interrupt driving power to the water pump.
AI & Schedule POST /api/v1/ai/trigger Manually trigger AI analysis using current physical sensor data and camera frame.
AI & Schedule GET /api/v1/schedule/upcoming Return the queue of planned future watering operations.
AI & Schedule GET /api/v1/schedule/history Retrieve a paginated collection of historical watering event executions.
AI & Schedule GET /api/v1/ai/suggestions Provide qualitative maintenance assessments and diagnostic alerts.
Configuration GET /api/v1/system/config Fetch global runtime parameters regulating hardware sampling loops.
Configuration PUT /api/v1/system/config Modify system operational variables (sensor intervals, cron, safety thresholds).
Camera Stream GET /api/v1/camera/stream Get live continuous video stream in standard MJPEG format.
